## ♻️ swwwcycle 
### custom wallpaper cycler for [swww](https://github.com/LGFae/swww)

![demo](./demo.gif)

### Dependencies

- swww

### Installation

- just grab the latest executable from the release page
- run the executable

### Building

- On NixOS
  - enter the dev shell with `nix develop`
  - run pyinstaller

- run pyinstaller:
  - `pyinstaller --onefile --name wallpaper-changer main.py`
